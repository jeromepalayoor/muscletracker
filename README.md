# Muscle Tracker

Muscle tracker is a simple workout tracker android app designed to log reps and track PRs.

## Build

1. Clone the repo
2. Open in Android Studio (Java, min SDK 28, compileSdk 35)
3. Run on a device or emulator

## Exercise data

Exercise database sourced from [free-exercise-db](https://github.com/yuhonas/free-exercise-db), formatted with additional specific-muscle, anatomical region, and split-category tags.
